# UVa Online Judge Solutions

:trophy: Solutions to problems from the UVa Online Judge :trophy:

## Usage

- If is a C++ file (.cpp):
```bash
g++ file.cpp -o main
./main
```

- If is a C file (.c):
```bash
gcc file.c -o main
./main
```

- If is a Java file (.java):
```bash
javac file.java
java file
```
